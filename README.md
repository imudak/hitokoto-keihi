# ひとこと経費記録 PWA

一言入力で経費を記録できる超シンプルPWA

## 特徴

- 🎤 **音声/テキスト入力:** 「コンビニ 450円 昼食」のように一言で記録
- 📱 **PWA対応:** オフラインでも動作、ホーム画面に追加可能
- 🤖 **自動カテゴリ推定:** 食費・交通費・通信費等を自動分類
- 📊 **月次サマリー:** カテゴリ別集計をビジュアル表示
- 💾 **ローカルストレージ:** データは端末内のみ（サーバー送信なし）

## 開発環境

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 使い方

1. 入力フォームに「場所 金額 メモ」を入力
   - 例: `コンビニ 450円 昼食`
   - 例: `タクシー 1200円`
   - 例: `Amazon 3500円 マウス購入`

2. 🎤ボタンをクリックして音声入力も可能

3. カテゴリは自動推定（編集可能）

4. 月次サマリーで支出を可視化

## 技術スタック

- React 18
- TypeScript 5
- Vite 5
- Tailwind CSS 3
- LocalStorage

## デプロイ

Cloudflare Pagesでデプロイ可能：

```bash
# ビルド
npm run build

# Cloudflare Pagesにデプロイ
# distディレクトリをアップロード
```
